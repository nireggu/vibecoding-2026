## 프로젝트: 업무 이력 관리시스템 

### 목표
- 일정 지연 없고, 누락없는 업무 진행을 위해 업무 진척률 및 진행 상황을 관리하는 업무 이력 관리시스템

### 기능
- 업무 내용 등록(제목, 내용, 요청일, 마감기한, 중요도, 요청자, 책임자, 작업상태)
- 업무 내용 수정
- 업무 내용 삭제
- 업무 일정 캘린더 표출
- 캘린더 업무 일정 클릭 시, 등록한 업무 게시물로 이동
- 업무 상태별, 업무 중요도별, 책임자별 대시보드에 건수 표출
 
### 디자인
- 화이트, 블루, 블랙 계열
- 깔끔하고 시원한 디자인

### 화면구성

#### 1. 메인화면
- 좌측에 대메뉴로 '메인', '일정관리' 구성
- 상단에는 업무 상태별, 업무 중요도별, 책임자별 건수를 표출할 수 있는 대시보드로 구성
- 하단 좌측에는 등록한 업무 목록을 게시판 형태로 볼 수 있도록 목록 구성
- 하단 우측에는 캘린더 배치

#### 2. 일정관리 화면
- 등록한 업무 내용을 확인할 수 있는 게시판 형태의 화면 구성
- 게시글 등록할 수 있는 버튼 추가
- 게시글 목록 선택 시, 업무 내용을 상세하게 확인할 수 있는 일정관리 상세화면 필요

#### 3. 일정관리 상세화면
- 일정관리 화면에서 선택한 게시글의 상세 내용을 확인할 수 있는 화면 구성
- 하단에는 내용을 수정하고, 삭제할 수 있는 버튼 필요
- 일정관리 화면(게시글 목록)으로 돌아갈 수 있는 목록 버튼 필요

### 기술
- HTML, CSS, JavaScript
- localstorage 사용
